プライバシーポリシー

ラーメンマップ

本アプリ「ラーメンマップ」（以下「本アプリ」）は、ユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、以下の情報を収集することがあります。 ・位置情報 　現在地周辺のラーメン店を検索するために使用します。位置情報はお客様の明示的な許可のもとでのみ取得します。 ・利用情報 　アプリの利用状況（画面閲覧、機能の使用など）を収集し、サービスの改善に役立てます。

2. 情報の利用目的

収集した情報は、以下の目的で使用します。 ・近隣のラーメン店の検索・表示 ・アプリの機能改善・不具合対応 ・利用傾向の分析によるサービス向上

3. 第三者サービスの利用

本アプリは以下の第三者サービスを使用しており、各サービスのプライバシーポリシーが適用されます。 ・Google Maps Platform（地図表示・店舗検索） ・YouTube Data API v3（動画情報の取得） ・Firebase / Google Analytics（利用状況の分析） ・Google AdMob（広告配信） 各サービスのプライバシーポリシーについては、Google のプライバシーポリシーをご参照ください。

4. 位置情報の取り扱い

位置情報は、現在地周辺のラーメン店を表示する目的にのみ使用します。位置情報を第三者へ販売・提供することはありません。位置情報の利用を停止したい場合は、端末の設定からアプリへの位置情報アクセスを拒否してください。

5. 情報の保存について

食べた店・行きたい店・レビューなどのデータはお客様の端末内にのみ保存されます。外部サーバーへのアップロードは行いません。

6. 広告について

本アプリはGoogle AdMobによる広告を表示します。AdMobはお客様の興味・関心に基づく広告を表示するため、広告識別子を使用することがあります。設定画面から広告の表示をオフにすることができます。

7. 子どものプライバシー

本アプリは13歳未満の方を対象としていません。13歳未満の方の個人情報を意図的に収集することはありません。

8. プライバシーポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合はアプリ内でお知らせします。

9. お問い合わせ

プライバシーポリシーに関するご質問は、アプリ設定画面の「お問い合わせ」よりご連絡ください。
